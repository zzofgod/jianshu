# jianshu

#### 介绍
 react高仿简书部分功能

  

#### 软件架构
- 项目用到了React全家桶、styled-conmponents类库、react-thunk中间件;
- 项目还用到了redux-immutable，immutable结合redux来管理数据，性能更好；
- 用json文件模拟数据，用axios来做ajax请求；
- 多次用到的功能都抽离成组件


#### 安装教程

1. npm i
2. npm start
